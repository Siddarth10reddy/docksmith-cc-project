## Overview

Docksmith is a simplified containerization system inspired by Docker, developed as part of a Cloud Computing mini project.

## Features

- Image building using Docksmithfile  
- Layer-based architecture  
- Cache mechanism (hit and miss)  
- Container execution  
- Environment variable support
